What Are the Functions on the Calculator?

  • /   Division
  • ×   Multiplication
  • +   Addition
  • −   Subtraction
  • =   Calculate
  • +/-   Plus/minus toggles the pos/neg sign of the displayed number
  • mc   Memory clear
  • mr   Memory recall
  • m-   Memory minus
  • m+   Memory plus
  • CE   Clear entry
  • AC   All clear
  • √x   Square root
  • %   Percentage
  • π   pi = 3.1415926536
  • xy   Exponent
  • R2   Round to 2 decimals (cents)
  • R0   Round to 0 decimals (dollars)
  • you can also use keyboard for complex calculation
